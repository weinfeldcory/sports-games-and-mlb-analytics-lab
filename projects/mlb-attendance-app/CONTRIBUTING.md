# Contributing

This repository is for private collaboration on Ballpark Ledger.

## Rules

- Do not commit secrets, passwords, tokens, or privileged backend credentials.
- Do not add third-party code, datasets, images, or assets without recording the source and license.
- Do not use MLB, team, league, or venue logos or other proprietary marks without permission.
- Treat all contributions as intended for assignment to the project owner or future company in writing.
- If you touch hosted auth or persistence, preserve user isolation and avoid any user-specific runtime behavior.
